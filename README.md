# cloud-architecture-portfolio
The portfolio focuses on designing resilient and scalable infrastructure solutions that integrate on-premises environments with cloud services. These projects demonstrate how enterprise workloads can be modernised while maintaining operational consistency and reliability.

Key areas explored include:

- Hybrid cloud architectures integrating on-premises infrastructure with AWS
- Container orchestration using Amazon ECS and ECS Anywhere
- Highly available multi-AZ infrastructure designs
- Secure Virtual Private Cloud (VPC) network architectures
- Managed database deployments using Amazon RDS
- Hybrid file storage using AWS Storage Gateway and Amazon S3
- Operational management and automation using AWS Systems Manager and AWS Backup
